### Deskripsi Singkat Merek

```
ZenUML adalah merek untuk layanan editor diagram UML berbasis web 
yang memungkinkan pengguna membuat, mengedit, dan berkolaborasi pada 
diagram UML secara real-time dengan dukungan multiple diagram types 
(Class, Use Case, Sequence, Activity, State, Component, Deployment, 
Package) dan export/import dalam berbagai format (SVG, PNG, PDF, XMI, 
StarUML).
```

## 8. Proses Pendaftaran

### 8.1 Tahapan Pendaftaran

#### Tahap 1: Persiapan (1-2 hari)
```
1. Kumpulkan semua dokumen yang diperlukan
2. Verifikasi kelengkapan dokumen
3. Buat salinan digital semua dokumen
4. Siapkan logo dalam berbagai format
5. Kumpulkan bukti penggunaan merek
6. Buat daftar barang/jasa yang jelas
```

#### Tahap 2: Pengajuan Permohonan (1 hari)
```
1. Datang ke Ditjen HAKI atau akses online (SIMHAKI)
2. Isi formulir permohonan pendaftaran merek
3. Serahkan semua dokumen
4. Bayar biaya pendaftaran
5. Dapatkan nomor pendaftaran sementara
6. Simpan bukti pembayaran
```

#### Tahap 3: Pemeriksaan Formal (1-2 minggu)
```
1. Ditjen HAKI memeriksa kelengkapan dokumen
2. Verifikasi data pemohon
3. Verifikasi logo dan daftar barang/jasa
4. Hubungi jika ada kekurangan dokumen
5. Terbitkan pengumuman pendaftaran
```

#### Tahap 4: Pemeriksaan Substantif (2-4 minggu)
```
1. Ditjen HAKI memeriksa kesamaan merek
2. Verifikasi tidak melanggar hak merek lain
3. Verifikasi tidak melanggar peraturan
4. Hubungi jika ada keberatan
5. Buat keputusan permohonan
```

#### Tahap 5: Penerbitan Sertifikat (1-2 minggu)
```
1. Jika disetujui, sertifikat diterbitkan
2. Dapatkan nomor registrasi resmi
3. Sertifikat berlaku 10 tahun
4. Dapat diperpanjang setiap 10 tahun
5. Simpan sertifikat dengan aman
```
